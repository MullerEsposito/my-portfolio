Dev News

Dev News é uma plataforma de consumo de conteúdo. Através do Prismic CMS, ele consome seu conteúdo usando a API chamada pela biblioteca axios. Existe um processo de autenticação usando a API do GitHub. É preciso estar autenticado para poder acessar o conteúdo completo das postagens. As credenciais de autenticação são armazenadas no Fauna DB. Também existe um sistema de pagamento por assinatura usando a API Stripe.

Máquina de Citações Aleatórias

Trata-se de um miniprojeto utilizando Javascript Vanila, HTML5 e SASS. O mesmo é capaz de gerar citações aleatórias a cada vez que clica no botão "New Quote". É possível também compartilhar a citação através de sua conta do Twitter.
